## 2. 개요
 - **Wordpress와 MySQL**로 구현한 **학과 블로그 사이트**입니다. 


 
## 3. 특징
- 워드프레스 자체의 플러그인을 이용하여 게시판을 구현하였습니다.
- 학과 수강과목과 커리큘럼, 일정에 관하여 검색기능을 구현했습니다.



## 4. 개발 환경 & 개발 툴
